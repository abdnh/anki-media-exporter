This add-on allows you to export media files from a chosen deck or note to a folder. The add-on can be accessed from different locations:
  • From the "Export Media" menu item under the gears icon besides each deck's name in the main screen.
  • From a button in the editor toolbar.
  • From the Notes > Export Media browser menu item to export from selected notes.
